# Real Estate Cash Flow Statistical Analyzer (C)

## Project Description
This project implements a statistical analyzer system in C for real estate financial data.  
The system reads Net Operating Income (NOI) values and computes statistical measures such as mean, median, and min/max.

## Features
- Manual data input
- File input (property_data.txt)
- Dynamic memory allocation (malloc, realloc)
- Mean calculation
- Median calculation
- Min / Max analysis

## File Structure
- main.c – program control
- menu.c / menu.h – menu interface
- data.c / data.h – data input
- fileio.c / fileio.h – file reading
- stats.c / stats.h – statistical functions
- property_data.txt – sample dataset


## Sample Data
The dataset represents monthly NOI values of a property.
